# iyaa CLI — Professional Project Bootstrapper

Overview - iyaa is a professional CLI tool to quickly scaffold frontend, backend, or full-stack projects using popular frameworks & languages.
With simple interactive prompts, you can generate boilerplate code for your next project in seconds.

## Features: 
 Interactive CLI interface
 Supports frontend, backend, and fullstack templates
 
 Multiple frontend frameworks:
 
  React (JavaScript / TypeScript)
  
  Next.js (JavaScript / TypeScript)
  
  Angular
  
  Vue.js
  
  Svelte
  
  SolidJS
  
  Astro
  
  Qwik
  
 Supports backend frameworks:
 
  FastAPI
  
  Django
  
  Express.js
  
  Nest.js

 Modular code structure for easy extension

 Fully installable via PyPI

## Installation:

Prerequisites: 

Python 3.7 or higher

pip installed

## Install from Pypi: `$ pip install iyaa`

## Usage: 
Once installed, simply run:

`$ iyaa init`

You’ll be guided through a series of prompts:

1️⃣ Enter project name

2️⃣ Enter directory where project should be created

3️⃣ Select project type: Frontend / Backend / Full Stack

4️⃣ Select framework (based on type)


5️⃣ Select language if applicable (TypeScript / JavaScript)

6️⃣ Enter author name

7️⃣ Project scaffold will be generated automatically
